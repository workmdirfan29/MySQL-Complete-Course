SQL (Structured Query Language) is a standard programming language used to manage and manipulate relational databases. With SQL, you can:

Create and modify database structures (tables, views, etc.)

Insert, update, delete data

Query data (e.g., find all users older than 18)

Control access to the data

1. Relational Databases (RDBMS)
Data is stored in tables (rows and columns)

Uses SQL

Examples:

MySQL

PostgreSQL

Oracle

Microsoft SQL Server

SQLite

2. NoSQL Databases
Stores data in formats like key-value, documents, graphs, or wide-columns

Designed for big data and real-time web apps

Examples:

MongoDB (Document-based)

Redis (Key-value)

Cassandra (Wide-column)

Neo4j (Graph)

3. NewSQL
Combines features of both SQL and NoSQL (scalability + SQL features)

Examples:

Google Spanner

CockroachDB

VoltDB
